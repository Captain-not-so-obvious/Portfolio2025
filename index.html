Sistema interno para Controle de frotas

Desenvolvi um sistema interno para empresa onde trabalho para controle da frota. Com React no front-end e Python/Django no back-end e com PostreSQL como SGBD, criei uma ferramenta que rastreia a posição dos veículos, monitora gastos e gera relatórios detalhados. Hospedado localmente por escolha estratégica, ele atende perfeitamente às nossas necessidades, provando que eficiência e simplicidade podem andar juntas.

Dream Team NBA

Projeto é uma aplicação web simples que permite ao usuário gerar o seu próprio "Dream Team" da NBA. Ao clicar no botão "Gerar Time", um script em JavaScript é acionado para exibir jogadores para as seguintes posições: armador, ala-armador, ala, ala-pivô e pivô. O projeto foi desenvolvido utilizando HTML para a estrutura, CSS para a estilização e JavaScript para a lógica de geração dinâmica do time.
